# Contributing

We welcome contributions to Meeting Intelligence Suite!

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Add tests
5. Submit a Pull Request

## Development Setup

```bash
git clone https://github.com/YOUR_USERNAME/meeting-intelligence-ai
cd meeting-intelligence-ai
npm install
npm test
npm run dev
```
